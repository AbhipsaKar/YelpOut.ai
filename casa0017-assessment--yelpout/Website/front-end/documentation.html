The Yelp business.json dataset was the input to the statistical model. The statistical analysis was performed in R using the dplyr, jsonlite, and tibble packages. Data cleaning and output to a csv format was performed in Python using the pandas package. Database is created and stored in tables using mysql. Bootstrap is used for header css styles, DeckGL is used for the map and Echarts, KJ Healy's US County map repository is used for the US state map. Canva was used to create the logo, mapsmarker and star mark font were used for the map markers. The API was written in Node.js with the Express package, with the mysql package used to communicate with the server, and the dotenv package used to reference auxiliary host, username, password, and database keys. Node Forever was used to keep the server-side Node.js code running.
